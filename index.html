To-Do List

To-Do List. This JavaScript project enables users to manage a list of todos with a simple and intuitive interface. Users can add new todos through a form and mark them as completed by clicking on them. The todos are stored locally using localStorage, ensuring persistence across page reloads. Additionally, users can remove todos by right-clicking on them. The project enhances productivity by providing a streamlined way to organize and track tasks.
